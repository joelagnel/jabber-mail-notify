Emacs Jabber mail notifier

Details:

Google's XMPP implementation has a nice extension to notify new mail which isn't ofcourse a part of the XMPP standard.
This little plugin extends the Emacs jabber client to support this.

Compatibility:

- Several changes have been made to make it compatible with the latest emacs-jabber (0.7.93)
- Make sure you have atleast a similar version (0.7 or higher)
- I'd appreciate it if you could report any bugs or issues with your version.

Installation Instructions

1. Download the file below, and copy it to your jabber directory. For me its ~/repository/list/el/net/jabber/
2. Open jabber.el in the same diretory and add the following line to the file:
(require 'jabber-mail-notify)
3. Restart emacs, or alternatively re-load the files jabber.el and jabber-mail-notify.el.
4. Have fun!

Usage

Simply connecting to your google talk account will check email the first time and will display a list of new unread mail in the minibuffer. All subsequent new mail notifications will be displayed in the minibuffer.

Download

Download jabber-mail-notify.el
